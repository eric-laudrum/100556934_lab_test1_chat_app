## Dependencies
bash
- npm install express socket.io nodemon
- npm install mongodb
- npm install dotenv

## Run the app
Navigate to root directory (where server.js is located)
bash
- npx nodemon server.js

## Using the app
- Create an account
    - Enter a Username, First Name, Last Name, and a Password
- Login
    - Enter your username and password to log into your new account
- Join a Chat Room
    - Select a chat room from the drop down menu and click the Join Room button
- Chat with other users
    - Type messages in the box below
    - Click send when you're ready to broadcast your message
    - When you're ready to leave the room, click the Leave Room button
